Lesson 13: Lists, Arrays, and Dictionaries

In programming, we often need to store multiple values together. Lists, arrays, and dictionaries help us manage data efficiently.

1. Lists (Python) and Arrays (JavaScript & Java)

Lists and arrays store multiple values in a single variable.

Python Example (Lists):

fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # Outputs: apple

JavaScript Example (Arrays):

let fruits = ["apple", "banana", "cherry"];
console.log(fruits[0]); // Outputs: apple

Java Example (Arrays):

String[] fruits = {"apple", "banana", "cherry"};
System.out.println(fruits[0]); // Outputs: apple

2. Dictionary (Python) and Objects (JavaScript)

Dictionaries store data in **key-value pairs**, making it easy to look up values.

Python Example (Dictionary):

person = {"name": "Vien", "age": 16}
print(person["name"])  # Outputs: Vien

JavaScript Example (Object):

let person = { name: "Vien", age: 16 };
console.log(person.name); // Outputs: Vien

3. Adding and Removing Elements

Python Example:

fruits.append("orange")  # Adds an item
fruits.remove("banana")  # Removes an item
print(fruits)

JavaScript Example:

fruits.push("orange"); // Adds an item
fruits.splice(1, 1); // Removes "banana"
console.log(fruits);

4. Why Use Lists, Arrays, and Dictionaries?

  • They help organize and manage data efficiently.
  • They allow easy retrieval and modification of data.
  • They are widely used in real-world applications (e.g., databases, APIs).

What is CI/CD?

CI/CD (Continuous Integration & Continuous Deployment) automates software testing and deployment.

1. Continuous Integration (CI)

CI automates the process of merging and testing code.

  • Developers push changes to a shared repository.
  • Automated tests run to check code quality.
  • Ensures that new code doesn’t break existing features.

2. Continuous Deployment (CD)

CD automates deployment so that every successful update is delivered to users.

3. DevOps Tools

  • Jenkins: Automates CI/CD pipelines.
  • Docker: Packages applications into containers.
  • Kubernetes: Manages containerized applications.

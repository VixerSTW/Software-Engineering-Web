Lesson 8: Understanding Variables and Data Types

Variables are used to store data in programming. Every programming language has different types of data that variables can hold.

1. What is a Variable?

A variable is like a container that stores a value. You can change the value at any time.

Example:

# Python
name = "Vien"
age = 16
print(name, age)

2. Common Data Types

Different types of data are used in programming. Here are some common ones:

Data Type Description Example
Integer (int) Whole numbers 10, -5, 1000
Float (float) Decimal numbers 3.14, -0.5, 99.99
String (str) Text "Hello", "Vien"
Boolean (bool) True or False values True, False

3. Declaring Variables in Different Languages

Python:

name = "Vien"
age = 16
is_student = True
print(name, age, is_student)

JavaScript:

let name = "Vien";
let age = 16;
let isStudent = true;
console.log(name, age, isStudent);

Java:

String name = "Vien";
int age = 16;
boolean isStudent = true;
System.out.println(name + " " + age + " " + isStudent);

4. Changing Variable Values

You can update a variable's value anytime:

# Python
x = 5
print(x)  # Output: 5
x = 10
print(x)  # Output: 10

Now that you understand variables and data types, let's move to **Lesson 9: Operators and Expressions**!

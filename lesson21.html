Modifying Data in SQL

SQL provides commands to insert, update, and delete data in a database.

1. INSERT INTO - Adding Data

                
INSERT INTO users (name, email) 
VALUES ('Vien', 'vien@example.com');

2. UPDATE - Modifying Existing Data

                
UPDATE users 
SET email = 'newemail@example.com' 
WHERE name = 'Vien';

3. DELETE - Removing Data

                
DELETE FROM users WHERE name = 'Vien';

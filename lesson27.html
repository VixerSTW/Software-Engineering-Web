What is Cloud Computing?

Cloud computing provides on-demand computing resources like servers, databases, and storage over the internet.

1. Types of Cloud Services

  • IaaS (Infrastructure as a Service): Provides virtual machines and storage (e.g., AWS EC2, Google Compute Engine).
  • PaaS (Platform as a Service): Provides a managed environment for app development (e.g., Heroku, Firebase).
  • SaaS (Software as a Service): Web-based applications (e.g., Google Drive, Gmail).

2. Web Hosting

Hosting is the process of storing a website on a server so it can be accessed online.

  • Shared Hosting: Multiple websites share the same server (e.g., Bluehost, Hostinger).
  • VPS Hosting: A dedicated portion of a server (e.g., DigitalOcean, Linode).
  • Cloud Hosting: Uses multiple servers for better reliability (e.g., AWS, Google Cloud, Vercel).

3. Deploying a Website

To deploy a static website, you can use services like:

  • GitHub Pages - Free hosting for static websites.
  • Vercel & Netlify - Easy deployment for frontend projects.

What is a SQL Transaction?

A transaction in SQL is a sequence of operations that are executed as a single unit. It ensures data integrity.

1. Using Transactions

                
BEGIN TRANSACTION;

UPDATE accounts SET balance = balance - 100 WHERE id = 1;
UPDATE accounts SET balance = balance + 100 WHERE id = 2;

COMMIT;  -- Saves the changes
ROLLBACK;  -- Cancels the changes

2. What is Indexing?

Indexes improve database performance by making searches faster.

3. Creating an Index

                
CREATE INDEX idx_users_email ON users(email);

This index makes searching by email much faster.

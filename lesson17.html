What is a Database?

A database is an organized collection of data that can be accessed, managed, and updated easily.

Types of Databases

  • Relational Databases (SQL) - Store data in tables with rows and columns.
  • NoSQL Databases - Store data in flexible formats such as JSON.

Common Databases

  • MySQL
  • PostgreSQL
  • MongoDB
  • SQLite

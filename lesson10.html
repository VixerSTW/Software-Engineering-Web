Lesson 10: Conditional Statements

Conditional statements allow programs to make decisions based on conditions. They help control the flow of execution in a program.

1. What are Conditional Statements?

Conditional statements check whether a condition is **true** or **false** and execute different code accordingly.

2. Types of Conditional Statements

1. if Statement

Executes a block of code if the condition is **true**.

# Python Example
age = 18
if age >= 18:
    print("You are an adult")
            
// JavaScript Example
let age = 18;
if (age >= 18) {
    console.log("You are an adult");
}

2. if-else Statement

Executes one block if the condition is **true**, otherwise executes another block.

# Python Example
age = 16
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")
            
// JavaScript Example
let age = 16;
if (age >= 18) {
    console.log("You are an adult");
} else {
    console.log("You are a minor");
}

3. if-elif-else Statement (Python only)

Checks multiple conditions.

age = 10
if age >= 18:
    print("You are an adult")
elif age >= 13:
    print("You are a teenager")
else:
    print("You are a child")

4. Switch Statement (JavaScript & Java)

A switch statement is used when checking multiple conditions.

// JavaScript Example
let grade = "B";
switch (grade) {
    case "A":
        console.log("Excellent!");
        break;
    case "B":
        console.log("Good job!");
        break;
    case "C":
        console.log("You can do better.");
        break;
    default:
        console.log("Invalid grade");
}

3. Why Use Conditional Statements?

  • They allow decision-making in programs.
  • They help create interactive applications.
  • They make programs more dynamic and flexible.

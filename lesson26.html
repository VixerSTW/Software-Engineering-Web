How Does the Frontend Communicate with the Backend?

The frontend (user interface) sends requests to the backend (server) to retrieve or update data.

1. REST API Communication

The frontend makes HTTP requests to backend APIs using **fetch** or **Axios**.

                
// Fetch data from backend (JavaScript)
fetch("https://api.example.com/data")
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error("Error:", error));

2. Sending Data from Frontend to Backend

To send data, use a POST request.

                
// Sending data with fetch API
fetch("https://api.example.com/data", {
  method: "POST",
  headers: {
    "Content-Type": "application/json"
  },
  body: JSON.stringify({ name: "Vien", age: 16 })
})
.then(response => response.json())
.then(data => console.log("Success:", data))
.catch(error => console.error("Error:", error));

3. Connecting Frontend and Backend in a Web App

A typical full-stack web application consists of:

  • Frontend: HTML, CSS, JavaScript (React, Vue, Angular).
  • Backend: Node.js, Python, Java.
  • Database: MySQL, MongoDB.

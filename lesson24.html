What is a RESTful API?

A RESTful API (Representational State Transfer) allows applications to communicate over the internet using HTTP requests.

1. HTTP Methods in REST APIs

  • GET: Retrieve data from the server.
  • POST: Send new data to the server.
  • PUT: Update existing data.
  • DELETE: Remove data from the server.

2. CRUD Operations

CRUD (Create, Read, Update, Delete) represents the basic actions used in database operations.

3. Example REST API Request

                
# Fetching data from an API (JavaScript)
fetch("https://api.example.com/users")
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error("Error:", error));

Lesson 15: Working with Files and Data

Many applications need to store and retrieve data from files. This lesson covers how to work with files in different programming languages.

1. Reading and Writing Files

Most programming languages allow you to open, read, and write files.

Python Example:

# Writing to a file
with open("data.txt", "w") as file:
    file.write("Hello, Vien!")

# Reading from a file
with open("data.txt", "r") as file:
    content = file.read()
    print(content)

JavaScript Example (Node.js):

const fs = require('fs');

// Writing to a file
fs.writeFileSync("data.txt", "Hello, Vien!");

// Reading from a file
let content = fs.readFileSync("data.txt", "utf-8");
console.log(content);

Java Example:

import java.io.*;

public class FileExample {
    public static void main(String[] args) throws IOException {
        // Writing to a file
        FileWriter writer = new FileWriter("data.txt");
        writer.write("Hello, Vien!");
        writer.close();

        // Reading from a file
        BufferedReader reader = new BufferedReader(new FileReader("data.txt"));
        System.out.println(reader.readLine());
        reader.close();
    }
}

2. Why Work with Files?

  • To **store and retrieve** user data.
  • To process **large datasets** efficiently.
  • To save settings and configurations for applications.

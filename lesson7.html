Lesson 7: Writing Your First Program

Now that your development environment is set up, it's time to write your first program! The classic first program is called **"Hello, World!"**, which simply prints a message to the screen.

1. Writing Your First Program

Below are examples of how to write "Hello, World!" in different programming languages.

Python:

print("Hello, World!")

JavaScript (Browser Console):

console.log("Hello, World!");

Java:

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

C++:

#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

2. Running Your Code

  • Python: Open a terminal and type: python script.py
  • JavaScript: Open the browser console (F12 in Chrome) and type: console.log("Hello, World!");
  • Java: Compile with: javac Main.java then run: java Main
  • C++: Compile with: g++ program.cpp -o program then run: ./program

3. Understanding the Output

Each of these programs will output:

Hello, World!

Now that you can run basic programs, you're ready to learn about **variables and data types** in the next lesson!

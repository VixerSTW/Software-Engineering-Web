Practice Your Skills

Below are coding exercises to help reinforce what you've learned.

1. Variables & Data Types

Write a program that declares a variable and prints its value.

                
# Example (Python)
name = "Vien"
print(name)

2. Conditional Statements

Write a program that checks if a number is positive, negative, or zero.

                
# Example (Python)
num = int(input("Enter a number: "))

if num > 0:
    print("Positive")
elif num < 0:
    print("Negative")
else:
    print("Zero")

3. Loops

Write a loop that prints numbers from 1 to 5.

                
# Example (Python)
for i in range(1, 6):
    print(i)

4. Functions

Write a function that takes two numbers and returns their sum.

                
# Example (Python)
def add(a, b):
    return a + b

print(add(3, 5))

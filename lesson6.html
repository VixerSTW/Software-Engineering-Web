Lesson 6: Setting Up Your Development Environment

Before you start coding, you need to set up a development environment that includes the right tools and software.

1. Choosing a Code Editor

A code editor helps you write and manage your code. Here are some great options:

  • VS Code - A popular, lightweight editor with great extensions.
  • Sublime Text - Fast and minimalistic, good for simple projects.
  • JetBrains IDEs - Great for professional developers (IntelliJ, PyCharm, etc.).

2. Installing a Compiler or Interpreter

Depending on the language you choose, you might need a compiler or an interpreter.

  • Python: Install Python from python.org.
  • JavaScript: Comes built into browsers, but you can use Node.js for backend development.
  • C/C++: Install GCC or Clang.
  • Java: Install the JDK (Java Development Kit).

3. Setting Up a Development Environment on Your Phone

If you're using a phone instead of a PC, here are some great apps for coding:

For Android:

  • AIDE - A great app for Java, C++, and Android app development.
  • Pydroid 3 - A Python IDE that allows you to run Python scripts.
  • Quoda - A lightweight code editor for multiple languages.
  • Termux - A terminal emulator that lets you run Linux commands, Python, and more.

For iPhone (iOS):

  • Pythonista - A powerful Python editor with built-in libraries.
  • Koder - A versatile code editor that supports many languages.
  • Textastic - A professional-grade code editor with syntax highlighting.
  • iSH - A Linux shell for running simple development tools.

4. Version Control with Git

Git is used to track changes in your code and collaborate with others.

Download Git from git-scm.com and learn the basics:

# Check if Git is installed
git --version

# Set up your name and email
git config --global user.name "Your Name"
git config --global user.email "your@email.com"

# Initialize a new Git repository
git init

5. Setting Up Your First Project

Once you have your tools ready, you can start coding! A simple first project is to create a basic **Hello World** program in your chosen language.

# Python Example
print("Hello, World!")

Now that your environment is set up, you’re ready to start coding! In the next lesson, we’ll write our first simple program.

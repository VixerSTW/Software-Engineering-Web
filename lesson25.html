Why is Authentication Important?

Authentication ensures that only authorized users can access certain parts of an application.

1. What is JWT (JSON Web Token)?

JWT is a compact, self-contained way of securely transmitting information between parties as a JSON object.

                
{
  "alg": "HS256",
  "typ": "JWT"
}

{
  "user": "Vien",
  "role": "admin"
}

2. What is OAuth?

OAuth is an authentication framework that allows users to log in using external services like Google, Facebook, or GitHub.

3. Example of JWT Authentication

                
// Creating a JWT token in Node.js
const jwt = require('jsonwebtoken');
const token = jwt.sign({ user: "Vien" }, "secret_key", { expiresIn: "1h" });

console.log(token);

Lesson 11: Loops and Iterations

Loops allow you to repeat a block of code multiple times. This is useful when working with large amounts of data or automating repetitive tasks.

1. Types of Loops

1. For Loop

The **for loop** is used when you know how many times you want to repeat an action.

Python Example:

for i in range(5):
    print("Hello, Vien!")

JavaScript Example:

for (let i = 0; i < 5; i++) {
    console.log("Hello, Vien!");
}

Java Example:

for (int i = 0; i < 5; i++) {
    System.out.println("Hello, Vien!");
}

2. While Loop

The **while loop** runs as long as a condition is **true**.

Python Example:

count = 0
while count < 5:
    print("Counting:", count)
    count += 1

JavaScript Example:

let count = 0;
while (count < 5) {
    console.log("Counting:", count);
    count++;
}

Java Example:

int count = 0;
while (count < 5) {
    System.out.println("Counting: " + count);
    count++;
}

3. Do-While Loop

A **do-while loop** runs at least once, even if the condition is **false**.

JavaScript Example:

let count = 0;
do {
    console.log("Counting:", count);
    count++;
} while (count < 5);

2. Why Use Loops?

  • To repeat actions multiple times.
  • To automate tasks (e.g., processing lists of data).
  • To make code **more efficient and readable**.

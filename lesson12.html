Lesson 12: Functions and Methods

Functions are reusable blocks of code that help organize and simplify your program. Methods are functions that belong to an object.

1. What is a Function?

A function is a block of code that only runs when it is called. Functions can take inputs (parameters) and return outputs.

Python Example:

def greet(name):
    return "Hello, " + name

print(greet("Vien"))

JavaScript Example:

function greet(name) {
    return "Hello, " + name;
}

console.log(greet("Vien"));

Java Example:

public class Main {
    static String greet(String name) {
        return "Hello, " + name;
    }

    public static void main(String[] args) {
        System.out.println(greet("Vien"));
    }
}

2. Function Parameters and Return Values

Functions can accept parameters (inputs) and return values (outputs).

Python Example:

def add(a, b):
    return a + b

print(add(3, 5))

JavaScript Example:

function add(a, b) {
    return a + b;
}

console.log(add(3, 5));

3. What is a Method?

Methods are functions that are tied to objects. For example, strings and lists have built-in methods.

Python Example:

text = "hello"
print(text.upper())  # Outputs: HELLO

JavaScript Example:

let text = "hello";
console.log(text.toUpperCase()); // Outputs: HELLO

4. Why Use Functions?

  • They make your code **reusable**.
  • They improve **readability and organization**.
  • They help reduce **repetition in code**.
